# Python dictionaries are ordered collectiuon of data items. They store multiple items in a single variable. Dictionary items are key-value pairs that are seprated by commas and enclosed within curly brakets{}.
info = {'name':'Karan', 'age':'19','eligible':True}
print(info)
Output :- {'name': 'Karan', 'age': '19', 'eligible': True}

# Accesing Dictionary items:-
I. Accesing single values: Values in dictionary can be accesed usiong keys. We can access dictionary values by mentioning keys either in squaue brakets or by using get method.
II. Accessing multiple values: We can print all the values in dictionary using values() methods. 
III. Accessing key-value pairs: We can print all keys in the dictionary using key() method.
IV. Accessing key-vale pairs: We can print all key-value pairs in the dictionary using items() method.
